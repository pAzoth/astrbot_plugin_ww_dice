# WW 无限规则骰点 · AstrBot 插件

World of Darkness / 无限恐怖风格骰池系统。支持加骰线、成功线、面数自由设定。

## 安装

将 `astrbot_plugin_ww_dice` 文件夹放入 AstrBot 的 `plugins/` 目录，重启或到面板加载插件。

## 用法

### 基础掷骰

```
.ww 10          → 10骰，默认 A10K8M10
.ww 10a5        → 加骰线=5（成功线沿用默认）
.ww 10a5k6m7    → 全指定：加骰线5、成功线6、7面骰
.ww20           → 无空格也支持
```

### 参数说明

| 参数 | 含义 | 默认 |
|------|------|------|
| `a` (加骰线) | ≥此值触发额外骰子 | 10 |
| `k` (成功线) | ≥此值计为成功 | 8 |
| `m` (面数)   | 骰子面数 | 10 |

a 和 k 独立判定：骰出 ≥a → 加骰；骰出 ≥k → 成功。

### 修改默认设定

```
.ww set k6       → 改成功线为6
.ww set a8k6m9   → 批量修改
.ww set clr      → 重置默认 (A10K8M10)
```

### 输出标记

- `[10]` — 加骰 + 成功
- `*8*` — 成功但不加骰
- `{5}` — 加骰但不成功（k > a 时出现）
- `4` — 失败

骰子 ≤30 时显示逐轮详情，超过则只显示统计摘要。
